# Tracker de Materias
Este proyecto es una aplicación web sencilla para llevar el control de las materias de la facultad. La armé principalmente para resolver una necesidad propia de organización y para poner en práctica conceptos de JavaScript aplicados a la manipulación del DOM y el manejo de almacenamiento en el navegador.

El proyecto está desarrollado con HTML, CSS y JavaScript vanilla, corriendo en un entorno Linux.

## Funcionalidades

- Registro de materias y actualización de su estado (En curso, Regular, Aprobada).
- carga de notas de parciales y finales.
- persistencia de datos mediante LocalStorage para que la información no se pierda al recargar la página.

## Tecnologías

-HTML5 y CSS3 para la estructura y el diseño.
- javaScript (ES6) para toda la lógica y la gestión de los datos.
